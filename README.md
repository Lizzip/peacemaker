# Peacemaker 
Code to run an LED strip for a Wynonna Earp Peacemaker

## peacemaker.py
Python code to be run on a Raspberry Pi 2B

## peacemaker.ino
C++ code to be run on an Arduino Nano

## Running
sudo python peacemaker.py
